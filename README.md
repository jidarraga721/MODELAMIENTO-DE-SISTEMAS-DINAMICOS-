# 🔬 Modelado de Sistema Térmico en MATLAB

# Resumen del proyecto

Este repositorio documenta el desarrollo de un laboratorio enfocado en la **identificación y modelado dinámico de un sistema térmico real**, en el marco del curso *Fundamentos de Control y Automatización*.

A partir de datos experimentales obtenidos en laboratorio, se construyeron distintos modelos matemáticos con el objetivo de representar el comportamiento del sistema y analizar su respuesta frente a una señal de entrada tipo escalón.

---

#Enfoque del trabajo

El análisis se basa en técnicas de **identificación de sistemas**, permitiendo obtener representaciones dinámicas a partir de datos reales.

Se desarrollaron los siguientes modelos:

- 🔹 Modelo de primer orden con tiempo muerto (FOPDT)  
- 🔹 Modelo de segundo orden  
- 🔹 Evaluación comparativa entre modelos  

Además, se realizó un análisis cuantitativo del ajuste mediante métricas de error.

---

#Procesamiento y simulación

Las simulaciones y el análisis se realizaron en MATLAB utilizando:

- Definición de sistemas dinámicos (`tf`)
- Simulación de respuestas (`step`)
- Visualización de resultados (`plot`)
- Toolbox de identificación de sistemas
- Evaluación de desempeño mediante:
  - Error cuadrático medio (ECM)
  - Raíz del error cuadrático medio (RMSE)

---

#Propósito del laboratorio

El desarrollo del proyecto tuvo como finalidad:

- Comprender el comportamiento dinámico de un sistema térmico real  
- Construir modelos matemáticos a partir de datos experimentales  
- Comparar la respuesta simulada con la respuesta real  
- Aplicar fundamentos de control automático en un contexto práctico
